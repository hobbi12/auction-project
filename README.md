BidSphere is a modern, real-time online auction platform where users can buy and sell unique items through exciting live bidding.
Built with cutting-edge technologies, BidSphere delivers a fast, secure, and visually appealing experience for both buyers and sellers.
✨ Key Features

Live Auctions with real-time bidding
Beautiful Dark UI with smooth animations
Secure Authentication using JWT + HttpOnly Cookies
Full Auction Management (Create, Edit, Delete)
Multiple Categories (Cars, Motorcycles, Real Estate, Electronics, Watches, etc.)
Responsive Design – works perfectly on mobile and desktop

🛠️ Tech Stack

Frontend: Next.js 16 (App Router) + TypeScript + Tailwind CSS
Backend: NestJS + TypeORM
Database: PostgreSQL (Supabase)
Authentication: JWT with HttpOnly Cookies

🎯 Goal
To create a simple yet powerful auction platform that provides an engaging and trustworthy environment for users to discover and bid on exclusive items.
